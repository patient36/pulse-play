# Pulse-Play Frontend

Pulse-Play is a music player application that allows users to listen to their favorite tracks, view lyrics, and interact with audio content. The frontend is built using React, providing a seamless and responsive user interface for browsing and playing music.

## Features

- Display a list of tracks with cover images, titles, and artists.
- Click on a track to view its details in a modal.
- Play audio with a built-in audio player.
- Display lyrics synchronized with the track.
- Modal with track details and audio controls.

## Technologies Used

- **React** - For building the user interface.
- **CSS** - For styling the application.
- **HTML5 Audio API** - For handling audio playback.
- **JSX** - For templating and UI structure.
- **React Hooks** - For managing state and lifecycle in functional components.

## Installation

To get started with the frontend of Pulse-Play, follow these steps:

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or above)
- [npm](https://www.npmjs.com/) (comes with Node.js)

### Steps to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/patient36/pulse-play.git
   ```

2. Navigate to the root directory of the repository:

   ```bash
   cd pulse-play
   ```

3. Navigate to the `client` directory:

   ```bash
   cd client
   ```

4. Install dependencies:

   ```bash
   npm install
   ```   

5. Start the development server:

   ```bash
   npm run dev
   ```

6. Open your web browser and navigate to [http://localhost:3000/](http://localhost:3000/).
